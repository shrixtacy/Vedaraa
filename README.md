# VEDARA - Interior Design Studio

A modern, minimalistic interior design website built with React, TypeScript, and Tailwind CSS.

## Features

- **Minimalistic Design** - Clean, elegant interface with gold accents
- **Responsive Layout** - Optimized for all devices
- **Smooth Animations** - Framer Motion powered interactions
- **Modern Architecture** - Built with Vite and TypeScript

## Pages

- **Home** - Hero section with services preview
- **About** - Company philosophy and approach
- **Portfolio** - Showcase of design projects
- **Catalogue** - Complete service offerings
- **Contact** - Contact form and information
- **Meeting** - Booking system for consultations

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Vite
- React Router

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Design System

- **Colors**: Deep black background with gold (#EBCB4B) accents
- **Typography**: Space Grotesk font family
- **Layout**: Consistent two-column structure (1/3 label, 2/3 content)
- **Animations**: Smooth transitions with cubic-bezier easing
